## Note: Hugging Face Spaces (IMPORTANT)

**Hugging Face Spaces is NOT a Next.js hosting platform.**

**Runtime provided by HF Spaces:**

* Python runtime
* Gradio / Streamlit
* Simple static HTML

**It is NOT:**

* A Node.js server
* A Next.js runtime
* A fullstack React environment

---

## So, what does HF Spaces actually host?

HF Spaces does **NOT** natively host Next.js apps.
HF Spaces hosts:

* Python-based AI applications (Gradio / Streamlit)
* Simple static frontends
* Backend logic + LLM / model inference

HF Spaces is suitable for **AI model demos** or **LLM playgrounds**,
**not** for deploying a production **Next.js application**.
